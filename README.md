
#  Readme-Generator

Built With:

* HTML
* CSS
* Javascript
* Node.js
* axios
* inquirer
* util

## Steps to install the project
1. Clone the repo on your system.
2. run command npm install in the project to install dependencies.
3. then run command node index.js to run the project.
